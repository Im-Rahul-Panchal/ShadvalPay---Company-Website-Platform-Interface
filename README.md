# ShadvalPay Website

A modern **React + Vite frontend website** built for the ShadvalPay fintech platform.  
The project includes a responsive UI with animations and a backend API for **form submissions using Node.js and SQL**.

# 🛠 Tech Stack

## Frontend
- React 19
- Vite
- TailwindCSS
- Radix UI
- Framer Motion
- React Router
- Lucide Icons
- Lottie Animations

## Backend
- Node.js
- Express.js
- SQL Database
- Form Submission API
